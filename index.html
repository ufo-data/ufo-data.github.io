UFO Sightings Geo-Map







We used Geo Map to see the sightings distribution across US. The size of red dots on the geo map is directly proportional to number of sightings in the that state. Geo map suggests more UFO sightings in coastal states such as california.

Use mousever on red dot to know the sightings count for each state.






We added the population information to see if states with more population have more sightings. The choropleth map shows the population using the color. Darker shade implies more population.

Source: U.S. Census Bureau, Population Division, 2016

Use mousever on red dot to know the sightings count for each state alog with its population.






We added the military base information to see if states with more military bases have more sightings. The choropleth map shows the number of military bases using the color. Darker shade implies more military installations.

Source: U.S. Military Bases, Bureau of Transportation Statistics

Use mousever on red dot to know the sightings count for each state alog with its military installation's count.

About

This is a classroom data visualization project to showcase the use of D3.js and other libraries (crossfilter.js , dc.js ) to create interactive graphics.


Team Members : Abhishek Kumar, Ritesh Soni, Lucas Dan

Data is extracted from National UFO Reporting Center website and then processed using Pandas.
